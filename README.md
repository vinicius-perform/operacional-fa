# FA Nexus - Operacional Hub

**FA Nexus** é uma plataforma operacional interna de alta performance, desenhada para centralizar o controle de registros diários e gestão de ações estratégicas com uma interface premium e experiência de usuário de nível mundial.

---

## 🎨 Sistema de Design Aura

O sistema utiliza o **Aura Design System**, focado em:
- **Glassmorphism**: Transparência e profundidade inspiradas em interfaces modernas (Apple/Linear).
- **Mobile First**: 100% responsivo, adaptando-se de desktops a dispositivos móveis sem perda de funcionalidade.
- **Micro-interações**: Animações fluidas que guiam o usuário através dos processos operacionais.

## 🚀 Funcionalidades Principais

### Hub Principal (Bento Grid)
- Visão geral consolidada dos registros operacionais.
- Métricas de eficiência e cartões de urgência para tomada de decisão rápida.

### Diário Operacional
- Registro detalhado de atividades por cliente.
- Sistema de badges de status dinâmicos (Excelente, Otimizando, Atenção, Iniciando).
- Controle de versão com data de última modificação sincronizada.

### Gestão de Ações (Lembretes)
- Lista de tarefas operacionais com prazos reais.
- Atribuição de responsáveis e níveis de prioridade.
- Modo de edição funcional para ajustes contínuos.

### Avançado (Admin Mode)
- Configurações de sistema e controle de fluxos internos.
- Segurança através de gate de autenticação persistente.

## 🛠️ Stack Tecnológica

- **Framework**: Next.js 16 (App Router)
- **Biblioteca**: React 19
- **Estilização**: Tailwind CSS v4 (Aura Upgrade Utility)
- **Animações**: Framer Motion
- **Ícones**: Lucide React
- **Persistência**: LocalStorage (v3)

## 📦 Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para gerar a build de produção:
   ```bash
   npm run build
   ```
